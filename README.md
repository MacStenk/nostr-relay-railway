# Nostr Relay on Railway

Personal Nostr relay for relay.stevennoack.de

Part of the Creator Freedom Stack.

## Setup

1. Railway: Deploy from GitHub
2. Environment Variables:
   - `RELAY_URL` - wss://relay.stevennoack.de
   - `RELAY_NAME` - Steven's Nostr Relay
   - `RELAY_DESCRIPTION` - Description
   - `RELAY_CONTACT` - steven@stevennoack.de
   - `VERIFY_ALL` - true
3. Custom Domain: relay.stevennoack.de
4. Deploy!

## After Deployment

Add to Nostr clients:

- Primal: Settings → Relays → Add
- URL: wss://relay.stevennoack.de
